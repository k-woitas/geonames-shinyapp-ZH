For running shinyapp "GeoNames Entities in ZH":
1. Open shinyapp.Rproj
2. Run app.R.

Accompanying files:
.RData - Environment which will be loaded by shinyapp.Rproj, contains raw and prepared ZH data, and also used functions
GeoNameEntities.R - Data preparation script
FUN_converting_geo_coordinates.R - Used geo coordinates transformation functions
